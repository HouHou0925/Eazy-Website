# Eazy-Website
Introduction
--

簡單的購物網站 MVC架構結合資料庫

請先安裝 SQL2017EXPRESS版本 (一定要EXPRESS 否則要去web.config修改設定)
開啟SQL->新建一個Item的資料庫->執行Script.sql->開啟專案
